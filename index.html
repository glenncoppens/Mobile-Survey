Appirio built a Mobile-Survey tool to demonstrate how easy it is to deploy a full web-app using the Heroku platform.

Mobile-Survey allows users to login and take surveys from mobile devices as well as from desktop browsers.

The web-app has 3 major components: 1. Heroku services (including Postgres, Heroku Connect) 2. Frontend web app 3. Salesforce backend

Surveys are built in Salesforce. Salesforce data is synchronized bi-directionally with Postgres DB using Heroku Connect. Restful web services transmit survey data between Postgres and the web-app.
